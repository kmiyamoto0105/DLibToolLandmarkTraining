# DLibToolLandmarkTraining
This Face Landmark updating tool is built in OS X using objective-c. It can parse Train.xml and open in GUI for updating landmark from http://dlib.net open source library. Those who are trying to predict shape in Human face can use this tool to edit input data set. Too learn more see Dlib face prediction.

Dlib face prediction is used to predict shapes of mouth, nose, eyes, face border etc. Dlib uses shape predictor class and deserialised
database to predict shape. If anyone wants to improve their data base using different dataset[Images], they need to learn the image and
regenerate the .dat file. This tool can draw 68 landmarks of face shapes in a corresponding image. it can also update the landmark file.
